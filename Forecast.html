Geographic Forecast

Using Machine Learning, this project forecast geographic conditions to improve safety for Singapore's aviation and maritime industry

Geographic Forecast

Skills: Data Analysis, Machine Learning, Python, SQL Database

View in Github Research Article

In this project we utilise multi-tasking modules to scrap real-time Geographic data of Singapore and store them in an SQL database.

Using Catboost Regression Model and cross validations, we conduct multi-variate prediction with these data to forecast future geographic conditions for different regions of Singapore and visualise them using the Singapore's map
