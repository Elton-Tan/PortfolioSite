AI storytelling

Using generative AI, this project generates a unique story by analysing the sentiment of the user's input

AI Storyteller

Skills:Natural Language Processing, Sentiment Analysis, Generative AI, Cloud Computing

↓ Click to try. Scroll below for mobile users! ↓

Test in Environment

In this project dialog flow is utilised to train the agent on different phrases such that it can identify the sentiments behind the user's input

Utilising AWS Lambda, the Python codes are hosted to generate a unique story using generative AI concepts and is webhooked to Dialogflow using AWS API gateway

Instructions:


  1. Click on 'test in environment' to show the AI chat-bubble (white chat-bubble). For mobile, you need to scroll to the bottom to view
  2. Type 'begin story' to start the process. You may restart by typing 'restart' at anytime or if errors are encountered
  3. Try to use conventional names like John or Jack as they are easier for the agent to identify.
  4. You may choose to add a composite to the hero's or villain's name such as naming the hero Jack as 'Jack the guardian'.
